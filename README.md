## 后端服务

- 基础支持-获取获取 Access token
- 基础支持-获取微信服务器 IP 地址

> 成为开发者后，用户每次向公众号发送消息、或者产生自定义菜单、或产生微信支付订单等情况时，开发者填写的服务器配置 URL 将得到微信服务器推送过来的消息和事件，开发者可以依据自身业务逻辑进行响应，如回复消息。

## 验证消息的确来自微信服务器

| 参数      | 描述                                                                                         |
| --------- | -------------------------------------------------------------------------------------------- |
| signature | 微信加密签名，signature 结合了开发者填写的 token 参数和请求中的 timestamp 参数、nonce 参数。 |
| timestamp | 时间戳                                                                                       |
| nonce     | 随机数                                                                                       |
| echostr   | 随机字符串                                                                                   |
